# Flash Flood Zone Mapping — Himachal Pradesh

Spatial analysis and machine learning-based flash flood susceptibility mapping across Himachal Pradesh. Uses terrain derivatives (slope, TWI, SPI, drainage density), rainfall intensity, land cover, and soil data to delineate high-risk zones using Random Forest and XGBoost ensemble models.

Run scripts in order via `uv run python scripts/<script>.py`. Full pipeline and methodology in `.claude/PLAN.md`.
